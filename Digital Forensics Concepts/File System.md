## file system
- scope of one partition
- defines:
	- how files and directories are named
	- how file data is stored and located
	- how metadata is managed
	- how free space is tracked
- has internal structures; e.g. FAT, inode
- OS-aware, OS-dependent
- examples:
	- FAT32
	- NTFS
	- ext4

reference:
- [Brian Carrier, 2005] "File System Forensic Analysis"
