## partition system
- aka partitioning scheme
- operates at disk level
- stored in fixed location on disk
- defines:
	- how a storage device is divided into partitions
	- where partition metadata is stored
	- how to find start and end of each partition
- examples:
	- MBR(DOS)
	- GPT

## partition
- contiguous range of sectors defined by partition system

- [Brian Carrier, 2005] "File System Forensic Analysis"
