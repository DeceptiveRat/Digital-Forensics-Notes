### 1. *Italic*
- file names:
	- *malware.exe*
	- *password.txt*
- tool names:
	- *Volatility*
	- *Autopsy*
- abbreviations:
	- *TSK*

### 2. `Codebox`
- code
- hexadecimal numbers:
	- `0xDEADBEEF`
- file path:
	- `C:\Users\Public\Windows Defender.exe`
- function name:
	- `GetMessageW`
- variable name in code:
	- `var1`
